; ЗДЕСЬ приведены СТАРЫЕ значения физических настроек ТФК, вполне возможно что к данной версии
; они все уже поменялись и в default.cfg совершенно другие.

; данные настройки надо вставлять в cfgs\КАРТА.cfg либо, если хотите использовать их
; по дефолту, вставьте их в cfgs\default.cfg после phys load.
; физические команды:
; phys load - загрузить дефолтную физику
; phys save - сохранить физику в phys.cfg
; phys info - вывести на экран список параметров
; phys <параметр> -узнать значение параметра
; phys <параметр> <значение> - установить значение параметра
; все величины времени - В ТИКАХ (1 секунда = 50 тиков)
; команды phys load и изменения параметров в режиме multiplayer не работают!!!!
; для посылки своей физики сервером требуется команда (у сервера) net_phys_sending 1
